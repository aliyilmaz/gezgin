# 🌍 Gezgin

Gezgin is a lightweight multilingual web application designed for tourism professionals.

---

## ✨ Features

- 📝 Notes management
- 💬 Phrase & translation management
- 📄 Report (minutes) creation
- 🏷️ Tagging and filtering system
- 📍 Location-based organization
- 🎤 Voice input and speech playback
- 🌐 Multi-language support (TR, EN, DE, FR, RU, AR)
- 💾 LocalStorage persistence (works offline)
- 🔄 Import / Export backup (JSON)

---

## 🎯 Purpose

To help tourism workers in the field to:

- Take quick notes  
- Communicate in different languages  
- Manage reports  
- Organize important information efficiently  

---

## 🧑‍💻 Tech Stack

- HTML5
- CSS3 (Bootstrap)
- Vanilla JavaScript
- Tagify

---

## 🚀 Setup

No installation required.

```bash
git clone <repo>
cd project
open index.html
```

## 📦 Data Storage
+ All data is stored in the browser (LocalStorage)
+ No backend required
+ Optional backup & restore support
