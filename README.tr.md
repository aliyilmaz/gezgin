# 🌍 Gezgin

Gezgin, turizm çalışanları için geliştirilmiş, çok dilli destek sunan hafif bir web uygulamasıdır.

---

## ✨ Özellikler

- 📝 Not yönetimi
- 💬 Çok dilli ifade ve çeviri yönetimi
- 📄 Tutanak oluşturma ve saklama
- 🏷️ Etiketleme ve filtreleme sistemi
- 📍 Lokasyona göre organize etme
- 🎤 Sesli not ve seslendirme
- 🌐 Çoklu dil desteği (TR, EN, DE, FR, RU, AR)
- 💾 LocalStorage ile veri saklama (offline çalışır)
- 🔄 JSON yedek alma ve geri yükleme

---

## 🎯 Amaç

Turizm sektöründe çalışan personelin sahada ihtiyaç duyduğu:

- Hızlı not alma  
- Yabancı dilde iletişim kurma  
- Tutanak tutma  
- Bilgileri düzenli saklama  

işlerini kolaylaştırmak.

---

## 🧑‍💻 Teknolojiler

- HTML5
- CSS3 (Bootstrap)
- Vanilla JavaScript
- Tagify

---

## 🚀 Kurulum

Herhangi bir kurulum gerektirmez.

```bash
git clone <repo>
cd proje
index.html dosyasını aç
```

## 📦 Veri Saklama
+ Tüm veriler tarayıcıda saklanır (LocalStorage)
+ Sunucu gerektirmez
+ İsteğe bağlı olarak yedek alınabilir
