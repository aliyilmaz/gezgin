## ✨ Overview

Gezgin is a lightweight, browser-based application designed for tourism professionals to manage:

- 📝 Notes
- 💬 Phrases & translations
- 📄 Reports (minutes)
- 🌐 Multi-language content

All data is stored locally in your browser (no backend required).

---

## 🚀 Features

- Multi-language support (TR, EN, DE, FR, RU, AR)
- Voice input & speech playback
- Tagging and filtering system
- Location-based organization
- LocalStorage persistence
- Import / Export (backup)

---

## 🧑‍💻 Tech Stack

- HTML
- CSS (Bootstrap)
